Overview

Thank you to everyone who has been reporting issues and sending feedback during the pre-launch. Every item in this update came from something you flagged. 🙏
No action needed to receive this update — it is live on the server already. If anything looks off, just send me a message or email. A bug reporting section is being worked on.

Version 1.2 is focused on easier navigation and QoL fixes. The focus has been on two things: making day-to-day workflow faster, and laying the groundwork for future features such as a proper dashboard.

Navigation & UI Overhaul

You will have noticed a clean new sidebar on the left side of the page, this is now how you navigate the app. You can expand it by clicking on the white space, or the little chevron at the top.

Once extended, you can see all the icon names, which are divided into sections. Click out of the sidebar, or press the chevron again to minimise it. You can also pin it open if you prefer.

All icons are clickable even in the minimal collapsed view. Tools that are greyed out are features currently in development and will be added in future updates.

The top-right corner has also had a small makeover: your account, sign out, and notification access are all in the same place as before, just cleaner.

Dynamic Pages

Pages now dynamically update their information. This means when you assign someone to a project, add submissions, log revisions, and more, the page will no longer do a full refresh.

This change is intended to make your experience smoother and keep you in context while you work.

All Projects Page

The all projects page has had a small but meaningful update. You can now filter projects (a search function is coming soon), and crucially, the grouping per CS lead has been removed.

All projects are now ordered by Design Deadline, with the closest deadline at the top, making it much easier to see what's urgent at a glance.

Email Notifications

Email notifications are now live and have undergone a visual overhaul to match the app's style. You can choose exactly which email notifications you want to receive in your My Account page. Toggle each event type on or off individually.

Bug Fixes

  • Fixed deadline time not showing on the project detail page
  • Fixed the Admin Panel being hidden behind the sidebar
  • Fixed Start Project not showing for some users
  • Fixed notifications for approved projects and submitted to client not firing

What's Next

Version 1.3 is already underway. The focus will be on the following:

  • File storage integration — Projects auto-creating folders, with links to that folder on the project detail page. Reference documents and submissions will also auto-save in the project folder on our local file storage.
  • Feature request page — Vote on features and track what's being worked on.
  • Bug report page — Report bugs with tracking on what's been acknowledged and is being worked on.
  • Dashboard — An overview of projects and designer load. Thanks for requesting this, Prim! 😁
  • Update blog — All future update posts like this one will live in-app, with a notification when a new update goes live.
  • Birthday Calendar — A calendar so we never forget anyone's birthday again 🥳. A global notification goes out one week before, and you'll be able to set your own birthday in My Account.
